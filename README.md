# Meta-Analysis Automation Engine

Bilimsel Veri İşleme Geleceğini Kuruyoruz 🚀

## 🎯 Misyon

Tıbbi literatürden otomatik veri çıkarabilen, doğrulama katmanları güçlü, 
hataya toleransı düşük bir meta‑analiz motoru geliştirmek.

## 📚 Ne Üzerinde Çalışıyoruz?

- PubMed API üzerinden makale toplama
- Bilimsel metinlerden otomatik veri çıkarımı (NLP + tablo parse)
- Halüsinasyon riskini azaltan çok katmanlı doğrulama
- Random‑effects ve mixed‑effects meta‑analiz modelleri
- Forest plot, funnel plot, heterojenlik analizleri
- Shapley değerleri ile moderatör katkı analizi
- Web tabanlı arayüz

## 👥 Aradığımız Ekip Arkadaşları

- Python'da güçlü olan
- NLP, PDF parsing veya backend geliştirme deneyimi
- Bilimsel doğruluk ve etik konularında hassas
- "Yaklaşık doğru" yerine "kanıtlanabilir doğru" prensibiyle çalışan
- Açık kaynak kültürüne yakın
